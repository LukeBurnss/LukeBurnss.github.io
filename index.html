Featured Projects

Honours Project

Role: Solo Developer Engine: Unity 2D
Unity C# AI Programming Boss Fight

My Honours project is a short demo game developed in Unity 2D, featuring a boss fight against an enemy which utilises behaviour trees for its AI. The player will have optional assist mechanics, to provide them help by automatically dodging, parrying or attacking enemies. The project aims to answer the research question: How can providing dynamic recommendations to player difficulty adjustments affect the overall difficulty of a game whilst maintaining flow? The project is in the early stages of development, and should be completed by the end of the academic year.

Pen & Sword

Team: Graphite Games Role: Gameplay Programmer Team Size: 8 Developers Context: Professional Project Engine: Unity 2D
Unity C# Hack n' Slash Tower Defence

Pen & Sword is a top down 2D hack n' slash meets tower defence style game developed using the Unity 2D game engine. I was one of three programmers in a team of eight, and focused on mechanics such as player and enemy movement and combat, as well as environmental interactions and the player radar. This project is a demo submitted for the DES315 module in my third year.
